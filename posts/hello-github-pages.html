GitHub Pages 블로그 시작하기

이 블로그는 별도의 빌드 과정 없이 GitHub Pages에서 바로 서비스할 수 있도록 구성했습니다. HTML 파일과 CSS 파일만 있으면 충분합니다.

글 추가 방법

새 글을 쓰려면 posts 폴더에 HTML 파일을 하나 만들고, index.html의 최근 글 목록에 링크를 추가하면 됩니다.

배포 방법

GitHub 저장소에 푸시한 뒤 저장소의 Settings → Pages에서 배포 소스를 Deploy from a branch, 브랜치를 main, 폴더를 /root로 선택하면 됩니다.

운영 팁

글이 많아지면 날짜, 주제, 요약을 기준으로 목록을 정리하고 필요할 때 Jekyll이나 Astro 같은 정적 사이트 생성기로 옮기면 됩니다.
